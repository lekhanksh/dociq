• CPU: 1 vCPU (1024 units)

Layer 4: Data Storage
RDS Aurora PostgreSQL

• Multi-AZ: Primary + Standby

Amazon S3

• Calculations support

• No data retention

✓ Production: Can use Claude 3.5 Haiku (cheaper)
Secrets Manager

CloudWatch

Layer 6: Networking & Security
VPC
• CIDR: 10.0.0.0/16
• 2 AZs (us-east-1a, us-east-1b)
• Public subnets: ALB
• Private subnets: ECS tasks
VPC Endpoints
• S3 (Gateway)
• Bedrock (Interface)
• Secrets Manager (Interface)
• ECR (Interface)
• CloudWatch Logs (Interface)
✓ No internet access needed
Security Groups
• sg-alb: 0.0.0.0/0:443 → ECS:8000
• sg-backend: ALB only → All out
• sg-rds: ECS only:5432
• Least privilege principle
