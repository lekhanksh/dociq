🤖 RAG Query Flow - Complete Pipeline

From Question to Answer: 11-Step Retrieval-Augmented Generation

⏱️ Total Time: 2-3 seconds per query

Query Flow Steps:

  1. User asks question - Natural language query submitted
  2. Detect complexity - Check for keywords: total, calculate, sum, all, across
  3. Generate embedding - Convert question to 384d vector
  4. Vector search - Find top 5-15 similar chunks (pgvector)
  5. Filter by department - Apply RBAC and multi-tenant isolation
  6. Group by document - Organize chunks by source file
  7. Build context - Format for AI with document boundaries
  8. Call Bedrock - Send to Amazon Nova Pro with system prompt
  9. Parse response - Extract answer with citations
  10. Audit log - Record query, user, IP, timestamp
  11. Return to user - Display answer with expandable sources

Key Features:

  • ✓ Multi-document reasoning (calculate across files)
  • ✓ Department-level isolation (HR can't see Finance)
  • ✓ Vector similarity search (50-100ms latency)
  • ✓ Source citations (every answer traceable)
  • ✓ Audit logging (compliance-ready)
  • ✓ Rate limiting (100 requests/min per user)
